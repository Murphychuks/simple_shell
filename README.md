#### Simple Shell

#### Usage:

- to compile use 
  - ```gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh```
- there are two way to use the shell
    - non interactive mode: ```echo "ls -la" | ./hsh```
    - interactive mode: ```./hsh```
-  we used valgrind to check leaks in our project
   -  ```valgrind ./hsh```
